Given Problem:

1. The code should able to take user input and give the merged output.

For example, input:

([49679, 52015], [49800, 50000], [51500, 53479], [45012, 46937], [54012, 59607], [45500, 45590], [45999, 47900], [44000, 45000], [43012, 45950])

Merged Result:

[43012, 47900], [49679, 53479], [54012, 59607]

===========================================================================

How to run the program (MergeOutput)?

Run Main Method:

1. Import the project into IDE eg. Eclipse
2. Run the main class -> MergeInputMainClass.java
   - Please enter the parameters from console ( total number of rows, columns & value for each element of array)
3. It will print the output in the console.

Run Test Cases:

1. Please run the JUnit test class -> MergeValuesTest.java
2. It will print the input values and output values in the console
3. JUnit Console shows green bar for the successful test case. ALso marks green for the method with successful test execution.
